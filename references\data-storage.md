# 数据格式参考

## 预览文件

- **文件名**：`greeting/YYYY-MM-DD-{001-999}.md`
- **存储位置**：由环境自动决定，只传相对文件名
- **生命周期**：生成预览 → 用户微调 → 读取发送 → 追加明细

### 预览内容格式

```markdown
### 发送前预览

| # | 候选人 | 发送内容 |
|---|--------|---------|
| 1 | 张大千 | 大千，节日快乐~ ... |
| 2 | 简致 | 简致，节日快乐~ ... |
```

### 明细格式（每发一人立即追加到同一文件，所有候选人都必须记录，含补发）

```markdown
### 发送数据明细

总 **N** 人 | 成功 **N** 人 | 失败 **N** 人 | 成功率 **X%**

| # | 候选人 | 发送内容 | 状态 | 时间 |
|---|--------|---------|------|------|
| 1 | 张大千 | 大千，节日快乐~ ... | 已发送 | 2026-03-12 10:36:54 |
| 2 | 简致 | 简致，节日快乐~ ... | 未找到 | - |
| 3 | 简致 | 简致，节日快乐~ ... | 已发送 | 2026-03-12 10:38:10 |
```

### 对话输出（文件链接放在明细表后面，文件中不写）

```
已完成，明细数据已更新: [greeting/YYYY-MM-DD-001.md](file://<实际绝对路径>)
```

## send 命令格式

```bash
# ✅ 正确：三参数，引号包裹
python3 scripts/main.py send "简致" "简致，植树节快乐~ ..." "植树节"

# ✅ 正确：日常关怀
python3 scripts/main.py send "简致" "简致，最近还好吗~ ..." "日常关怀"

# ❌ 错误：缺引号
python3 scripts/main.py send 简致 简致，植树节快乐 植树节

# ❌ 错误：参数顺序反了
python3 scripts/main.py send "简致，植树节快乐~" "简致" "植树节"

# ❌ 错误：缺少参数
python3 scripts/main.py send "简致" "植树节"
```

参数顺序：`send "名称" "内容" "主题"`，三参数必填。
